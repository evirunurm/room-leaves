### Stack

- Front-End
    - **HTML5**, **CSS3**, Javascript **ES6**
    - **Vue 3** SPA + **Vue Router** + Babel
    - **Axios**
    - **Aframe** + AR.JS + Mind-AR (3D Augmented Reality Graphics, in progress)
    - Hosted on **Netlify**


- Back-End (<a href="https://github.com/evirunurm/room-leaves-api">GitHub repo</a>)
    - Javascript, **NodeJS**
    - **Express** 4 + **JsonWebToken** (RESTful API)
    - **MySQL** 2 + **Sequelize** ORM
    - **Bcrypt** (Hash generation and validation)
    - Hosted on **Heroku**, ClearDB MySQL (Ignite) add-on
